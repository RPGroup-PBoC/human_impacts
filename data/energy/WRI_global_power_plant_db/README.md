
# WRI Global Power Plant DB

## Description
This is a non-exhaustive database of global power plants extracted from the WRI github repository.

## Key Numbers

* Global natural gas extraction (2019) ≈ 4.0 trillion cubic meters
* Global oil extraction (2019) ≈ 5.5 billion cubic meters
* Global coal extraction (2019) ≈ 8.1 billion tonnes

## Source Information

* **Source**: World Resource Institute
* **URL**: [WRI GitHub Project](https://github.com/wri/global-power-plant-database)
* **Original License**: As per the GitHub repo: "The latest database release (v1.2.0) is available in CSV format here under a Creative Commons-Attribution 4.0 (CC BY 4.0) license."
* **Bias**: Uncertain. Global non-profit with a long list of foundation and corporate funders. See their [donors](https://www.wri.org/about/donors).
